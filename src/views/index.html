Project Manager is a website developed by MAE to have all the resources required for the project in one location.

Features:

Create a website that enables the team members to track the progress of the project and also to have all their needs into one website. Allowing the team members to have all the resources required for the project. The members are registered into the website if only the admin sends them an invitation. Only the admin is allowed to add, edit and delete new courses, lessons and topics. The admin can track if the members completed a certain course, lesson and topic or not. Only the admin can add, edit, and delete resources from the website. The admin can send emails to the team members. The website shows notification for any new activity. Each topic has instructor notes, additional links, files and discussion forum to allow the users to discuss the topics.
